Metal Miners

Dynamic Health System

My health system can be used on any damageable object imaginable. The health system uses an abstract class named "Damageable" that consists of a few virtual methods. This structure allows for easy adaprtability to any situation involving destruction or combat. Every object in the game that the player can attack uses this script as a base and slightly changes it to fit the objects needs. For example, a crate object may run the base "Damage" method but, run some custom code as well. So a player and a crate both have the "Damageable" script but, the crate does slightly different things when its destroyed compared to the player.

Resource Spawning

The resource spawning system is built to allow easy implementation of new maps into the game. There is a "Resource Controller" that exists in the scene and it spawns resources in any available "Resource Spawn Zone". Each Resource Spawn Zone finds the Resource Controller and tells it that it exists. The Resource Controller simply iterates through the list of all its registered Resource Spawn Zones and spawns a rock object at a random location. This system is extremely efficient and is easily expandable. It also allows for creative freedom in the placement of the zones and the amount of zones.

Ability System

The ability system is remarkably expandable and powerful. There is a base abstract class called "Ability" which contains some abstract methods. An ability can be created by simply deriving from the Ability class and overriding the abstract methods to do whatever the ability desires.
